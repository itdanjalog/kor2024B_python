- 파일 처리
    - 파이썬 파일이 아닌 외부 파일( txt, json, csv , excel 등등 )를 읽고 쓰기
    - open( '파일의경로' , '열기모드' , encoding='인코딩종류' ) : 지정한 경로의 파일를 객체로 반환 하는 함수
        1. 파일의경로
            - '파일명.확장자'
                - 확장자 :
                    txt : 윈도우 메모장의 확장자
            - 절대경로 : 모든 경로
                C:\프로젝트가위치한모든PC경로\kor2024B_python\day11
                예] C:\Users\admin\Documents\hyunsu\kor2024B_python\day11

            - 상대경로 : 현재 코드가 위치한 경로 기준으로 지정한 경로를 찾아가는 방법
                ./ : 현재 파일의 위치 뜻한다.
                    step3.py 파일의 위치 = day11 폴더
                    './test1.txt' -> day11 폴더의 test1.txt

                ../ : 현재 파일의 한칸 위 위치 뜻한다.
                    step3.py 파일의 위치 = day11 폴더
                    '../test2.txt' -> kor2024B_python 폴더의 test2.txt
        2. 열기모드 :
            w:쓰기(존재하면 덮어쓰기 아니면 파일생성)
            r:읽기

        3. encoding : 파일의 문자를 인코딩(번역 하는) 방식 선택
            - 한글 인코딩은 cp949 , utf-8 주로 사용한다.

    - 파일객체
        - open() 함수의 반환 되는 객체
        - 주요 함수
            1. .write( '내용' )  : 파일내 내용 쓰기
            2. .close( )        : 파일 닫기
            3. .readline()      : 파일내 모든 내용을 한줄씩 읽어 오는 함수
            4. .readlines()     : 파일내 모든 내용을 한줄씩 요소로 해서 여러줄을 리스트로 반환 함수
            5. .read()          : 파일내 모든 내용을 문자열 로 읽어오는 함수
