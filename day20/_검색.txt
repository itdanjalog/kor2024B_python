[ 검색/탐색 ]
    - 목록 에서 특정한 값을 찾는 행위 # 색인
    - 검색/탐색 종류
        1. 순차/선형 검색
            - 리스트/배열의 첫번째 요소부터 차례대로 하나씩 값을 비교하면서 원하는 값을 찾는 방법
            - 정렬 여부와 상관없이 모든 데이터에 사용 할수 있다.
            - 예] 전화번호부에서 첫번째부터 차례대로 특정 사람 찾는 방법 유사

        2. 이진/이분 검색
            - 정렬된 배열/리스트 에서 중간값을 기준으로 원하는 값이 더 크면 오른쪽,더 작으면 왼쪽 으로 탐색 하는 방법
            - 정렬이 필수 이고 탐색 범위가 절반으로 줄여든다.
            - 예] 정렬된 책 목록에서 중간을 기준으로 찾아가면서 특정 책을 빠르게 찾는 방법

        3. 이진 트리 검색(day18)
            - 이진 트리 구조 에서 사용하는 방법
            - 각 노드가 두개의 자식 노드를 가지며, 왼쪽 자식 노드는 부모 노드보다 작고 오른쪽 자식노드는 부모 노드보다 크다.
            - 예] 트리 구조를 사용해 디렉토리에서 파일 탐색하는 방법

        4. 해시 검색(day07-2 딕셔너리)
            - 데이터를 고유한 해시 값으로 변환해 빠르게 접근할 수 있도록 하는 검색 방법
            - 해시 테이블을 이용하여 데이터와 그에 해당하는 해시 값을 저장하고 해시 값으로 데이터를 검색을 하는 방법
            - 예] 도서관에서 책 제목을 해시 함수로 변환해 저장하고 해시 값을 이용하여 책제목을 빠르게 검색하는 방법
                - '재미있는 파이썬' -- 해시함수 이용한 책 제목을 --> 해시값 변환 하여 저장
                - 변환된 해시값을 이용하여 다시 책 제목을 탐색/검색하는 방법 ,  해시테이블 구조 { 해시값 : 원본데이터 }
