'''
    웹 페이지(웹사이트) 크롤링
        - 크롤링 : 웹 페이지에서 데이터를 자동으로 수집하는 과정
        - 크롤링 또는 스크래핑 이라는 불린다.
        - HTML언어 구조를 분석 하고 필요한 정보 추출 해서 분석,저장 등등
        - 왜 ? 인터넷 의 방대한 정보를 수집 하고 , 효율적 으로 분석 , 자동화
        - 필요한 학습 : 파이썬 , 웹기술(HTML)

    HTML : Hyper Text Markup Language
        - 하이퍼 텍스트 마크업 언어 ( - 파이썬/자바/C언어 와 같이 웹 프로그래밍 언어의 종류 중 하나 )
        - 웹 페이지 제작 용도로 사용 되는 언어
    Markup : 마크업
        - 정해진 문법/약속 , 브라우저(크롬,엣지) 가 기능을 이해 할수 있도록 미리 정해 놓은 문법
        - 마크업 의 요소 구성 : < 꺽은괄호 를 이용한 문법을 구성한다.
            1. < 마크업이름 > 내용물 < /마크업명 >
            2. < 마크업 />
        - 마크업 종류
            1. <html > </html>
            2. <head> </head>
            3. <title>  </title>
            4. <h1> ~ <h6>
            7. <p> ~ </p>
            8. <div> ~ </div>
            7. <span> ~ </span>
            8. <ul> 안에 <li>
             그외 많은 HTML 마크업이 존재한다.
             - 하나의 마크업들을 여러번 작성할수 있으므로 마크업의 식별자(구분이름)를 부여한다.
                1. class 속성
                    <마크업 class="클래스명" > </마크업>

                2 .id 속성
                    <마크업 id"아이디명" > </마크업>

    HTML 파일 만들기
        1. 패키지를 오른쪽 클릭
        2. 새로만들기 -> html 파일 -> 1_웹페이지기초
